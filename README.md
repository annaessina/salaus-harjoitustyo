## Käyttöohje

1. Kloonaa projekti

2. Asenna riippuvuudet komennolla

```
poetry install
```
3. Käynnistä ohjelma komennolla

```
poetry run invoke start
```

4. Testit voidaan suorittaa komennolla

```
poetry run invoke test
```
